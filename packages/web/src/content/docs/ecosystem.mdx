A collection of community projects built on openpatent.

:::note
Want to add your openpatent related project to this list? Submit a PR.
:::

You can also check out [awesome-openpatent](https://github.com/awesome-openpatent/awesome-openpatent) and [openpatent.cafe](https://openpatent.cafe), a community that aggregates the ecosystem and community.

## Plugins

| Name                                                                                              | Description                                                                                    |
| ------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------- |
| [openpatent-helicone-session](https://github.com/H2Shami/openpatent-helicone-session)                 | Automatically inject Helicone session headers for request grouping                             |
| [openpatent-skills](https://github.com/malhashemi/openpatent-skills)                                  | Manage and organize openpatent skills and capabilities                                           |
| [openpatent-type-inject](https://github.com/nick-vi/openpatent-type-inject)                           | Auto-inject TypeScript/Svelte types into file reads with lookup tools                          |
| [openpatent-openai-codex-auth](https://github.com/numman-ali/openpatent-openai-codex-auth)            | Use your ChatGPT Plus/Pro subscription instead of API credits                                  |
| [openpatent-gemini-auth](https://github.com/jenslys/openpatent-gemini-auth)                           | Use your existing Gemini plan instead of API billing                                           |
| [openpatent-antigravity-auth](https://github.com/NoeFabris/openpatent-antigravity-auth)               | Use Antigravity's free models instead of API billing                                           |
| [openpatent-google-antigravity-auth](https://github.com/shekohex/openpatent-google-antigravity-auth)  | Google Antigravity OAuth Plugin, with support for Google Search, and more robust API handling  |
| [openpatent-dynamic-context-pruning](https://github.com/Tarquinen/openpatent-dynamic-context-pruning) | Optimize token usage by pruning obsolete tool outputs                                          |
| [openpatent-websearch-cited](https://github.com/ghoulr/openpatent-websearch-cited.git)                | Add native websearch support for supported providers with Google grounded style                |
| [openpatent-pty](https://github.com/shekohex/openpatent-pty.git)                                      | Enables AI agents to run background processes in a PTY, send interactive input to them.        |
| [openpatent-shell-strategy](https://github.com/JRedeker/openpatent-shell-strategy)                    | Instructions for non-interactive shell commands - prevents hangs from TTY-dependent operations |
| [openpatent-wakatime](https://github.com/angristan/openpatent-wakatime)                               | Track openpatent usage with Wakatime                                                             |
| [openpatent-md-table-formatter](https://github.com/franlol/openpatent-md-table-formatter/tree/main)   | Clean up markdown tables produced by LLMs                                                      |
| [openpatent-morph-fast-apply](https://github.com/JRedeker/openpatent-morph-fast-apply)                | 10x faster code editing with Morph Fast Apply API and lazy edit markers                        |
| [oh-my-openpatent](https://github.com/code-yeongyu/oh-my-openpatent)                                  | Background agents, pre-built LSP/AST/MCP tools, curated agents, Claude Code compatible         |
| [openpatent-notificator](https://github.com/panta82/openpatent-notificator)                           | Desktop notifications and sound alerts for openpatent sessions                                   |
| [openpatent-notifier](https://github.com/mohak34/openpatent-notifier)                                 | Desktop notifications and sound alerts for permission, completion, and error events            |
| [openpatent-zellij-namer](https://github.com/24601/openpatent-zellij-namer)                           | AI-powered automatic Zellij session naming based on openpatent context                           |
| [openpatent-skillful](https://github.com/zenobi-us/openpatent-skillful)                               | Allow openpatent agents to lazy load prompts on demand with skill discovery and injection        |
| [openpatent-supermemory](https://github.com/supermemoryai/openpatent-supermemory)                     | Persistent memory across sessions using Supermemory                                            |
