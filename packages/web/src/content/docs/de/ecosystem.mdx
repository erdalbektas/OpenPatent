Eine Sammlung von Community-Projekten, die auf OpenPatent basieren.

:::note
Möchten Sie Ihr OpenPatent-bezogenes Projekt zu dieser Liste hinzufügen? Senden Sie einen PR.
:::

Sie können sich auch [awesome-OpenPatent](https://github.com/awesome-OpenPatent/awesome-OpenPatent) und [OpenPatent.cafe](https://OpenPatent.cafe) ansehen, eine Community, die das Ökosystem und die Community zusammenfasst.
