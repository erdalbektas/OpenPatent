ชุดโครงการชุมชนที่สร้างขึ้นบน OpenPatent

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ OpenPatent ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-OpenPatent](https://github.com/awesome-OpenPatent/awesome-OpenPatent) และ [OpenPatent.cafe](https://OpenPatent.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน

---
