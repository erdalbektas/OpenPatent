Una colección de proyectos comunitarios construidos en OpenPatent.

:::note
¿Quieres agregar tu proyecto relacionado con OpenPatent a esta lista? Envíe un PR.
:::

También puedes consultar [awesome-OpenPatent](https://github.com/awesome-OpenPatent/awesome-OpenPatent) y [OpenPatent.cafe](https://OpenPatent.cafe), una comunidad que agrega el ecosistema y la comunidad.
