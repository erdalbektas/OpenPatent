## プラグイン

| [OpenPatent-openai-codex-auth](https://github.com/numman-ali/OpenPatent-openai-codex-auth)                    | API クレジットの代わりに ChatGPT Plus/Pro サブスクリプションを使用する                                               |
| [OpenPatent-gemini-auth](https://github.com/jenslys/OpenPatent-gemini-auth)                                   | API 課金の代わりに既存の Gemini プランを使用する                                                                     |

| [OpenPatent-devcontainers](https://github.com/athal7/OpenPatent-devcontainers)                                | 浅いクローンと自動割り当てポートを使用したマルチブランチ devcontainer の分離                                         |
| [OpenPatent-google-antigravity-auth](https://github.com/shekohex/OpenPatent-google-antigravity-auth)          | Google Antigravity OAuth プラグイン、Google 検索のサポート、およびより堅牢な API 処理                                |
| [OpenPatent-dynamic-context-pruning](https://github.com/Tarquinen/OpenPatent-dynamic-context-pruning)         | 古いツールの出力を削除してトークンの使用を最適化する                                                                 |
| [OpenPatent-websearch-cited](https://github.com/ghoulr/OpenPatent-websearch-cited.git)                        | Google ベースのスタイルでサポートされているプロバイダーにネイティブ Web 検索サポートを追加                           |
| [OpenPatent-pty](https://github.com/shekohex/OpenPatent-pty.git)                                              | AI エージェントが PTY でバックグラウンドプロセスを実行し、インタラクティブな入力を送信できるようにします。           |
| [OpenPatent-shell-strategy](https://github.com/JRedeker/OpenPatent-shell-strategy)                            | 非対話型シェルコマンドの手順 - TTY に依存する操作によるハングの防止                                                  |
| [OpenPatent-wakatime](https://github.com/angristan/OpenPatent-wakatime)                                       | wakatime で OpenPatent の使用状況を追跡する                                                                            |
| [OpenPatent-md-table-formatter](https://github.com/franlol/OpenPatent-md-table-formatter/tree/main)           | LLM によって生成された Markdown テーブルをクリーンアップする                                                         |
| [OpenPatent-morph-fast-apply](https://github.com/JRedeker/OpenPatent-morph-fast-apply)                        | Morph Fast apply API と遅延編集マーカーにより 10 倍高速なコード編集                                                  |
| [oh-my-OpenPatent](https://github.com/code-yeongyu/oh-my-OpenPatent)                                          | バックグラウンドエージェント、事前構築された LSP/AST/MCP ツール、厳選されたエージェント、Claude Code 互換            |
| [OpenPatent-notificator](https://github.com/panta82/OpenPatent-notificator)                                   | OpenPatent セッションのデスクトップ通知とサウンドアラート                                                              |
| [OpenPatent-notifier](https://github.com/mohak34/OpenPatent-notifier)                                         | 許可、完了、エラーイベントのデスクトップ通知とサウンドアラート                                                       |
| [OpenPatent-zellij-namer](https://github.com/24601/OpenPatent-zellij-namer)                                   | OpenPatent コンテキストに基づいた AI による自動 Zellij セッション命名                                                  |
| [OpenPatent-skillful](https://github.com/zenobi-us/OpenPatent-skillful)                                       | OpenPatent エージェントがスキルの検出と挿入を使用してオンデマンドでプロンプトを遅延ロードできるようにする              |
| [OpenPatent-supermemory](https://github.com/supermemoryai/OpenPatent-supermemory)                             | スーパーメモリを使用したセッション間での永続メモリ                                                                   |
| [@plannotator/OpenPatent](https://github.com/backnotprop/plannotator/tree/main/apps/OpenPatent-plugin)        | 視覚的な注釈とプライベート/オフライン共有による対話型の計画レビュー                                                  |
| [@openspoon/subtask2](https://github.com/spoons-and-mirrors/subtask2)                                     | OpenPatent/コマンドをきめ細かいフロー制御を備えた強力なオーケストレーションシステムに拡張                              |
| [OpenPatent-scheduler](https://github.com/different-ai/OpenPatent-scheduler)                                  | launchd (Mac) または systemd (Linux) を cron 構文で使用して、定期的なジョブをスケジュールする                        |
| [micode](https://github.com/vtemian/micode)                                                               | 構造化されたブレインストーミング → 計画 → セッション継続性のあるワークフローの実装                                   |
| [octto](https://github.com/vtemian/octto)                                                                 | 複数の質問フォームを使用した AI ブレインストーミング用のインタラクティブなブラウザ UI                                |
| [OpenPatent-background-agents](https://github.com/kdcokenny/OpenPatent-background-agents)                     | 非同期委任とコンテキスト永続性を備えた Claude Code スタイルのバックグラウンドエージェント                            |
| [OpenPatent-notify](https://github.com/kdcokenny/OpenPatent-notify)                                           | OpenPatent のネイティブ OS 通知 – タスクがいつ完了したかを知る                                                         |
| [OpenPatent-workspace](https://github.com/kdcokenny/OpenPatent-workspace)                                     | バンドルされたマルチエージェントオーケストレーションハーネス – 16 コンポーネント、1 回のインストール                 |
| [OpenPatent-worktree](https://github.com/kdcokenny/OpenPatent-worktree)                                       | OpenPatent 用のゼロフリクション Git ワークツリー                                                                       |
