Une collection de projets communautaires construits sur OpenPatent.

:::note
Vous souhaitez ajouter votre projet lié à OpenPatent à cette liste ? Soumettez un PR.
:::

Vous pouvez également consulter [awesome-OpenPatent](https://github.com/awesome-OpenPatent/awesome-OpenPatent) et [OpenPatent.cafe](https://OpenPatent.cafe), une communauté qui regroupe l'écosystème OpenPatent.
